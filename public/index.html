gtch calendars

I publish some calendars for myself and friends here. They work in calendar apps like Apple Calendar, Google Calendar, Fantastical, etc. You're welcome to use them too but you do so at your own risk — I provide the calendars without warranty of any kind.

Calendar URLs

  • Ambulance Victoria roster cycle: updates daily with the next 20 roster cycles
    https://calendars.gtch.au/roster-cycles.ical

Instructions

Apple Calendar (iPhone)
tap Calendars → Add Calendar → Add Subscription Calendar… then paste the URL into the Subscription URL box
Fantastical
go to File → New Calendar Subscription… then paste the URL into the Calendar URL box
Google Calendar
go to Other calendars → Add other calendars → From URL… then paste the URL into the box
